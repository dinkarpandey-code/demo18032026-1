# About Me Portal

**Created by Agent**

A personal portfolio website built with React + TypeScript + Vite, deployed to AWS S3 + CloudFront.

## Features
- Personal bio section with photo and bio
- Skills with proficiency bars grouped by category
- Project portfolio with category filtering
- Experience timeline
- Contact form with client-side validation
- Dark mode toggle with localStorage persistence
- Fully responsive design

## Tech Stack
- React 18 + TypeScript
- Vite build tool
- Vitest + React Testing Library for testing
- AWS S3 + CloudFront for hosting

## Scripts
```bash
npm install      # Install dependencies
npm run dev      # Development server
npm run build    # Production build
npm run test     # Run tests
npm run preview  # Preview production build
```
